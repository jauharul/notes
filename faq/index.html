Frequently Asked Question

virtualenv Bagaimana cara membuat virtual environment yang baru? 

mkvirtualenv <name> --python=<python-installation-directory>
Contoh:
mkvirtualenv flask --python=/home/jauharul/.local/bin/python2.7

Note: tambahan opsi --python berarti menggunakan instalasi selain default.

python Bagaimana menginstall python-package dari local directory? 

pip install --no-index <package-installation-file>
Contoh:
pip install --no-index Flask-0.9.tar.gz

Note: --no-index hanya memeriksa local directory. Jika tidak ada, tidak mendownload dari pypi.

bash Bagaimana membuat symbolic link? 

ln -s {$target-filename} {$symbolic-filename}
Contoh:
ln -s /home/jauharul/src/djangobook/_build/html/ /home/jauharul/public_html/djangobook

Note: Dengan symbolic link, akses dokumentasi menjadi mudah meski tidak di public_html.

heroku Bagaimana me-rename app? 

heroku apps:rename NEWNAME
Contoh:
heroku apps:rename quiz-ai-pendtium

Note: Cara lain adalah dengan mengubah melalui web interface. Jika berhasil, update heroku remote URL dengan git remote set-url heroku <newurl>.
